# wetube

https://nomadcoders.co/wetube

### Two Project Domains : [User] & [Video]

### Design URL

##### Global Router

- / : Home (Video)
- /signup : Sign Up (User)
- /login : Login (User)
- /search : Search (Video)
  <br>

##### User Router

- /user/:id : See User
- /user/logout : Log Out
- /user/edit : Edit MY Profile
- /user/delete : Delete MY Profile
  <br>

##### Video Router

- /video/:id : See Video
- /video/:id/edit : Edit Video
- /video/:id/delete : Delete Video
- /video/upload : Upload Video
